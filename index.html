About this Application.

The repository contains the IOS project developed using swift programming language for 301CEM coursework. The application focuses on providing users to choose trending movies and tv shows and is using TheMovieDB api for fetching the details combined with the Firebase .

Applied Swift Concepts

  • UICollectionView with Flow Layout
  • UITableView
  • AVPlayer
  • Core Data for data persistence
  • Programmatic Constraints.
